   - Format: JPG hoặc PNG

2. **project1.jpg** (800x500px)

   - Screenshot hoặc mockup của dự án 1
   - Format: JPG hoặc PNG

3. **project2.jpg** (800x500px)

   - Screenshot hoặc mockup của dự án 2
   - Format: JPG hoặc PNG

4. **project3.jpg** (800x500px)

## Nguồn lấy ảnh miễn phí (nếu chưa có):

- **Unsplash**: https://unsplash.com
- **Pexels**: https://pexels.com
- **Pixabay**: https://pixabay.com

## Tạo ảnh placeholder online:

- **Placeholder.com**: https://placeholder.com
- Ví dụ: `https://via.placeholder.com/350x350` cho profile
- Ví dụ: `https://via.placeholder.com/800x500` cho projects

## Lưu ý:

- Tối ưu hóa kích thước file (nên < 500KB/ảnh)
- Sử dụng công cụ như TinyPNG để nén ảnh
- Đảm bảo ảnh có chất lượng tốt và phù hợp với theme website
